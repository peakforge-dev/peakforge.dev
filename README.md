# peakforge.dev

Website for [Peakforge](https://peakforge.dev) — independent software studio building simple apps and useful tools.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to `main` — Vercel deploys automatically.
